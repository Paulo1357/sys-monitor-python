# System Monitor Python
Este é um script de automação em Python desenvolvido durante minha mentoria de Analista de Sistemas.

## 🚀 Funcionalidades
- Coleta dados de hardware e kernel do Linux.
- Registra logs de saúde do sistema com timestamp.
- Execução em ambiente isolado (venv).

## 🛠️ Tecnologias
- Python 3
- Git & GitHub
- Linux (KDE Plasma)
